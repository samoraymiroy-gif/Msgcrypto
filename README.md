# 🔐 Python Encryption Tool

A simple Python encryption and decryption tool that uses a user-defined key to generate a shift value for a Caesar-style cipher. It supports English and Arabic text, numbers, and symbols, and converts encrypted output into numeric format.

## Features

* Encrypt and decrypt text
* Key-based shift generation
* Supports English and Arabic characters
* Converts encrypted text to numeric (ASCII/Unicode) values
* File-based input and output

## Installation

Clone or download this repository to your computer:

```bash
git clone https://github.com/samoraymiroy-gif/Msgcrypto.git
```

Or download the project as a ZIP file and extract it.

## Usage

1. Write the message you want to encrypt in `message.txt`.
2. Open a terminal in the project directory.
3. Run the program:

```bash
python3 Msgcr.py
```

4. Choose whether to encrypt or decrypt.
5. Enter your secret key when prompted.

### Files

* `message.txt` — Input message.
* `encrypted.txt` — Encrypted output.
* `decrypted.txt` — Decrypted output.
